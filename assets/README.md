\# assets/



This directory stores \*\*raw input data\*\* used by the JobRunner workflow.



Contents include:

\- CFTC futures position data (ZIP format)



These files are treated as \*\*immutable inputs\*\*:

they are not modified by scripts.



⚠️ Note:

\- Data sources remain the property of their original providers.

\- Redistribution follows the original public-use policies.



CFTC data is sourced from the U.S. Commodity Futures Trading Commission

and is considered public domain.
